Full Experimental Configuration

The following describes all degradation-prior configurations used in this work. These settings constitute a sensitivity analysis of the DIL framework rather than a fully blind deployment scenario. The choice of h1[n] is fixed across all configurations for reproducibility.

DILo Oracle

h2[n] = h1[n]. The further-degraded signal y′[n] is generated using the same RIR as the observed y[n]. Internal training pairs are fully consistent with the true reverberation process, providing a best-case learning scenario. Serves as an upper-bound reference for the DIL framework.

DILε10 / DILε20 / DILε40 Synthetic

Controlled synthetic mismatch. h2[n] = h1[n] + e[n], where the variance of the error vector e[n] is set to 10%, 20%, or 40% of the variance of h1[n] respectively. This controls the overall energy of the estimation error. Quantifies sensitivity to calibrated perturbations of the degradation prior.

DILSR Same Room

Measured h2[n] from the same ACE room as h1[n] but at a different microphone position (independently drawn from the remaining four channels). h1 is randomly selected per utterance; selections are recorded for reproducibility. Although h2 and h1 share the same room environment and T60, they correspond to different microphone positions, introducing local spatial mismatch.

DILCR Cross Room

Measured h2[n] from a different ACE room with a similar T60. The same h1 index list is reused. To maintain simplicity, h2 is fixed to the first channel among the five available. For Lecture Room 2, no ACE room with a comparable T60 exists — results are marked N/A.

DILCD Cross Dataset

h2[n] drawn from the MIT Acoustic Survey [Traer & McDermott, 2016], a single-channel external dataset, selecting the RIR whose T60 most closely matches h1 for each room. The same h1 index list is reused. This configuration tests robustness to a fully external, unmatched prior.
