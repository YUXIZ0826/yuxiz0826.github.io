Experiment Design

🎯 Objective

Evaluate the DIL framework under noisy reverberant conditions (babble noise at SNR = 10 dB), assessing robustness beyond the clean-speech setting. Methods compared: WPE, DILo (oracle), DILSR (same-room mismatch).

🔊 Noise Source

CHILDREN_STUDENTS-talking.flac
Original: 44.1 kHz, stereo, 122.5 s.
Pre-processing: first channel → mono, resampled to 16 kHz (~1.96M samples).
SNR: 10 dB, applied per utterance with a reproducible random seed.

⚙️ Signal Generation

For each utterance i:
1. Convolve clean speech x[n] with h1 → reverberant y[n]
2. Draw noise segment using seed = utterance_idx, random start point within the 122.5 s file
3. Scale noise to target SNR = 10 dB and add: ynoisy[n] = y[n] + scaled noise
Start sample recorded in manifest for exact reproducibility.

🧠 DIL Training

Input: ynoisy[n] (noisy reverberant)
Pseudo pair: (h2 ∗ ynoisy, ynoisy)
Training target: ynoisy[n] (noise-inclusive)
Inference output: x̂[n] (estimated clean)

DILSR: h2 drawn from same ACE room, different channel (indices stored in h2_indices_SR.npy).

Full Experimental Configuration

The following describes all degradation-prior configurations used in this work. These settings constitute a sensitivity analysis of the DIL framework rather than a fully blind deployment scenario. The choice of h1[n] is fixed across all configurations for reproducibility.

DILo Oracle

h2[n] = h1[n]. The further-degraded signal y′[n] is generated using the same RIR as the observed y[n]. Internal training pairs are fully consistent with the true reverberation process, providing a best-case learning scenario. Serves as an upper-bound reference for the DIL framework.

DILε10 / DILε20 / DILε40 Synthetic

Controlled synthetic mismatch. h2[n] = h1[n] + e[n], where the variance of the error vector e[n] is set to 10%, 20%, or 40% of the variance of h1[n] respectively. Quantifies sensitivity to calibrated perturbations of the degradation prior.

DILSR Same Room

Measured h2[n] from the same ACE room as h1[n] but at a different microphone position (independently drawn from the remaining four channels). Selections are recorded for reproducibility. Introduces local spatial mismatch while sharing T60.

DILCR Cross Room

Measured h2[n] from a different ACE room with a similar T60. h2 is fixed to the first channel among the five available. For Lecture Room 2, no ACE room with a comparable T60 exists — results are marked N/A.

DILCD Cross Dataset

h2[n] drawn from the MIT Acoustic Survey [Traer & McDermott, 2016], selecting the RIR whose T60 most closely matches h1 for each room.
